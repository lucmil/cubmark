Detailed Description

Definition class.

Class for tokenizing parameter strings. See the constructor documentation for input syntax.

Constructor & Destructor Documentation

cb::Definition::Definition ( const std::string &  definitionStr)

Parse a definition string.

A definition string is of the form:

name:parameter_list

An example of a test function definition could be: W3:n=3,C_x=2

A parameter list is of the form:

parameter_1,parameter_2,...,parameter_n

where each parameter is a pair of name and value

name[=value]
