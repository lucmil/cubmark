Detailed Description

template<int DIM = 2>
class cb::RuleChain< DIM >

Rule chain for multi-dimensional quadrature product integration.

NOTE: This class doesn't support more than 2 dimensions at this moment.

However it could be easily extended to handle N dimensions if the function type of cb::Method::IntegrandType were to be modified to use arrays. Then only evaluateIntegrand would require modififcation.
