Detailed Description

Simple log manager.

Small logging class with the ability of logging messages of 4 different priorities and choice of output stream.

The default output stream for logs is std::cerr.

The user interface consists of three macros: LOG, LOG_STREAM, LOG_LEVEL

Member Enumeration Documentation

enum cb::LogManager::LogLevelType

Available message priorities The available log levels are (in order of priority):

  • LVL_FATAL - unrecovarable error which usually aborts the program
  • LVL_WARN - recovarable error which affects
  • LVL_INFO - general, terse run-time information
  • LVL_TRACE - detailed debugging information

Default logging priority level is LVL_INFO
