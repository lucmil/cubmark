Detailed Description

template<class FO, class R, class... A>
class cb::AdapterTraits< FO, R, A >

Adapter traits.

Use a template specialization of AdapterTraits to override this value.

For example, this snippet will allow you to use two different instances of the cb::Constant functor simultaneously via two adapters

namespace cb
{
template<>
class AdapterTraits<Constant>
{
public:
enum LimitsType { SLOT_LIMIT = 2 };
};
}
...
void somefunction()
{
cb::Constant A(1), B(1);
cb::BinaryAdapter<Constant>::FtorType a, b;
cb::adapter(a, A);
cb::adapter(b, B);
(a->getPtr())(0, 0);
(b->getPtr())(0, 0); // Would segfault if SLOT_LIMIT == 1
}
