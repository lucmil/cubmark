Detailed Description

Tracing shim for concrete Method classes.

Class for debugging concrete Method classes. Each function: a) logs arguments passed to it b) checks preconditions c) calls the concrete method. d) checks postconditions

If any checks fail, an exception will be thrown. Preconditions and postconditions for each function are mentioned in the Method class documentation.
